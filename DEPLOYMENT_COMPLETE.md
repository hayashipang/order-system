## 🚀 部署流程

### 本地開發
```bash
npm run dev          # 使用 SQLite，適合本地開發
```

### 雲端部署
```bash
# 1. 設定 Railway PostgreSQL
# 2. 部署後端到 Railway
railway up

# 3. 部署前端到 Vercel  
vercel --prod

# 4. 設定環境變數
# Railway: DATABASE_URL
# Vercel: REACT_APP_API_URL
```
